# 🎵 Spotify Clone Web App

A modern Spotify-inspired music streaming web application built using HTML, CSS, and JavaScript. This project replicates the basic UI and functionality of Spotify, allowing users to play music and explore songs.

---

## 📌 Overview

This project is a frontend clone of Spotify that demonstrates UI design and basic music player functionality. It is designed to improve frontend development skills and create a real-world project for portfolio use.

---

## 🚀 Features

* 🎧 Play and pause music
* 📃 Display list of songs
* 🎨 Modern UI inspired by Spotify
* 📱 Fully responsive design
* ⏱️ Music progress control (if added)

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
